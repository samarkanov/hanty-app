Intro

Description:  the clients (boxes below) can subscribe, unsubscribe to/from a particular event ("SendMessage", "ChangeColor"). You as a user can make subscriptions for clients. Once client is subscribed, it starts receiving (frequent) notifications leading to a change in the client's state. That state is preserved in a database. Meanwhile, UI keeps requesting the state of each client and make it visible here
